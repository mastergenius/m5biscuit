# Practical French Service Communication

Status: seed course source, 2026-05-18.

This course is for practical adult French use, not school-style A1 memorization. The first target is
service communication: booking an appointment, explaining a simple preference, repairing
misunderstanding, and leaving the interaction with enough control.

## Learner Goal

Build useful French sentence patterns for everyday service situations. The current concrete scenario
is a haircut appointment, because it forces the learner to combine booking, time slots, preferences,
degree words, and repair phrases.

## Current Boundary

The learner should be able to:

- build polite request frames with `je voudrais`;
- add time and service slots without overloading grammar;
- ask simple questions with `Est-ce que`;
- set practical boundaries like `pas trop court`;
- recover when comprehension fails by asking for repetition or demonstration;
- assemble a short personal script for a real service visit.
